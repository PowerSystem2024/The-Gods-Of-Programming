# Agencia Publicitaria

Este es un programa para gestionar anuncios publicitarios.

## Funcionalidades

- Mostrar precios
- Agregar, eliminar y modificar anuncios
- Buscar anuncios por empresa
- Calcular ingresos totales

## Requisitos

- JDK 11 o superior
- Compilador de Java
